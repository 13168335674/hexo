[推荐]Vue-devtools安装

vue-devtools是一款vue浏览器开发者工具调试插件

  1. 首先从github上将vue-devtools clone到本地

    git clone https://github.com/vuejs/vue-devtools.git

  2. 安装项目所需要的包

    npm install

  3. 编译文件

    npm run build

4.添加至游览器
